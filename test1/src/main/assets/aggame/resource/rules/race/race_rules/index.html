百家乐比赛

基于最受欢迎的百家乐游戏，迎合当前最具吸引力的比赛方式，以公平透明的比赛规则，为玩家奉上最具娱乐性的在线百家乐比赛。

锦标赛

到开赛时间，报名人数>=最低开赛人数即开赛。报名人数上限高，竞技性更强，奖金更丰厚。
若到开赛时间，报名人数<最低开赛人数，比赛将自动取消并退还用户报名费用（该比赛用户使用门票报名，则退还门票；使用现金报名，则退还现金）。

01 必须为AG真钱账户

02 必须持有相应比赛门票或余额不低于报名费用

03 点击 “进入赛场” ，进入赛场报名比赛

比赛条款及细则如下：

  1. 每个玩家持有相同数量的初始筹码。该比赛筹码与玩家真钱额度无关，仅用于比赛投注及排名，比赛结束后清空
  2. 比赛下注规则：
    • 1.每局必须下注，超时未下注则系统默认以最小限红代下闲
    • 2.每局必须下庄/闲，且不能同时下庄闲
    • 3.每局下庄/闲后，才可下注其它玩法
    • 4.筹码低于最低限红时，仅允许以All in形式下庄/闲
    • 5.每个玩法的下注，不可超出其限红范围（玩家可在界面右下角“游戏资料”中查看玩法限红）
  3. 比赛过程中，每个玩家均可使用一定使用次数的PASS和暗注功能：
    • PASS：跳过该局下注
    • 暗注：属于策略投注技巧，添加筹码到下注位后，点击“暗注”按钮即可进行暗注下注。使用暗注后，在该局停止下注时，其他玩家才能看到自己所下注的玩法及金额
  4. 晋级/淘汰机制：

    • 在多个轮次的比赛中，按每个轮次的晋级方式进行轮次更迭，共分为两种晋级方式。

      • 1、全场排名
        当前轮次为全场排名时，取当轮结束时指定排名前若干名的玩家晋级，其余玩家将自动淘汰。
        当前轮次若剩余玩家=晋级人数时，将直接进入下一轮比赛。
        示例：参赛人数100人，第一轮全场排名，前49名晋级。玩家随机分布于15个虚拟桌上进行全场竞技，当轮结束时排名前49名玩家晋级，其余淘汰。
      • 2、单桌排名
        当前轮次为单桌排名时，取当轮结束时每桌指定前若干名玩家晋级，其余玩家将自动淘汰。
        当前轮次若某一桌剩余玩家=每桌晋级人数时，该桌将不再开局，晋级玩家需等待轮次结束。
        示例：参赛人数49人，当前轮次单桌排名，每桌第1名晋级。玩家分布于7个虚拟桌上进行每桌拼杀。轮次结束时，每桌第1名晋级，其余淘汰； 若当前轮次局数未打完，某一桌只剩1个玩家，则该玩家收到晋级提示，需等待轮次结束。

    • 注意：可点击比赛详情查看每场比赛的赛制流程规则。

  5. 比赛排名规则：
    1. 比赛结束排名

      按玩家剩余的筹码数从高至低进行排名。不存在相同名次情况：

      • a) 筹码数相同的玩家，按胜率的高低排名；
      • b) 筹码数、胜率均相同的玩家，按有效投注额的高低排名；
      • c) 筹码数、胜率、有效投注额均相同的玩家，按报名时间的先后排名
    2. 淘汰排名方式
      • a) 同一局多个玩家被淘汰，依次按胜率高低、有效投注额多少、报名时间先后进行排名。
      • b) 比赛过程中 因筹码数为0淘汰的玩家，按被淘汰时间（先者靠后）的方式排名。
  6. 比赛特殊情况：

    比赛未达到指定局数，但出现以下任一情况时，将提前结束比赛，按照上文第4节中的排名规则决定名次：

    a) 只剩一个玩家未被淘汰；

    b) 所有玩家均被淘汰。

  7. 比赛结束后，AG奖金赛的奖金即时到账，无需等待和审核。赌神奖金赛、赌神总决赛的奖金将在比赛结束后的两个工作日内发放。
