游戏概述

骰宝也叫赌大小，是中国古老相传的游戏。这个游戏的用具是个密封的骰盅，由各玩家选择筹码下注，猜测经机械摇动后骰子开出的点数或是点数总合。

游戏玩法

1、 本游戏使用3颗骰子摇骰。

2、 开始新局后即开始下注倒数计时，您可以依照您的猜测，选择筹码下注 。

3、 倒数时间结束后停止下注，再由荷官按钮经机械自动摇骰。

4、 待骰盅停止后，视三颗骰子停留开出的点数，由荷官输入三点数，同时画面亮起灯光，可清楚看到胜出注码和赔率；是否与玩家押注的内容相同，来判定输赢。

点数计算

* 桌台限红是本游戏桌台主要玩法的最小、最大玩法限红。玩家可通过限红选择切换不同桌台限红。

* 骰子摇动后，偶有稳定后无法判读点数状况时，此时荷官将重摇骰，直到能够完整判读(如遇故障因素将注销所有相关注单/派彩)。

* 如遇结算错误，按照当期视频结果重新结算。
